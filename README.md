Streams twitter messages to web browsers in real time using web sockets.

## Dependencies:

* [node.js][1]
* [fab v3][2]
* [node.ws.js][3]
* [twitter-node][4]

## Usage

    $ user=NAME password=SECRET node server.js

For more info see my [blog post][5] on the subject.
